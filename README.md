## Getting Started

### Prerequisites

- Node.js installed

### Installation

1. Clone the repository:

    ```sh
    git clone git@github.com:ckroeger/leaflet-maps-made-easy.git
    cd leaflet-maps-made-easy
    ```

2. Install the dependencies:

    ```sh
    npm install
    ```

### Development

Start the live server and watch for changes:

```sh
npm run dev
```

### Build

Build an bundle:

```sh
npm run build
```
